# Azure AI Search Overview

Azure AI Search is a managed search and retrieval service used to ground AI applications in private or curated content. In this project it is treated as backing infrastructure for a Foundry knowledge base rather than as a custom application-owned retrieval index.

The portfolio target uses Azure Blob Storage for source files and Microsoft Foundry Agents with knowledge base or file search tooling for grounded answers. Azure AI Search supports retrieval behind that knowledge experience.
