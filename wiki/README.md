# GenAIOps Support Copilot Wiki

This wiki is the project memory for the GenAIOps Support Copilot portfolio build.

The project demonstrates how to design, build, evaluate, monitor, and operate a Microsoft Foundry Agents application grounded in Azure AI Search documentation. The target architecture uses Foundry knowledge base / file search grounding, Azure Blob Storage, Azure AI Search-backed retrieval, GitHub-based prompt management, Foundry evaluations, monitoring, and tracing.
